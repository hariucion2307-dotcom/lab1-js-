1.Теоретическая часть:
JavaScript — это язык программирования, используемый для создания интерактивных веб-страниц. Он выполняется в браузере (например, Google Chrome) или в среде Node.js (Node.js).
JavaScript позволяет:
объявлять переменные
использовать условия и циклы
работать с вводом пользователя
подключать внешние файлы
2.Формулировка задачи
1.Выполнить JavaScript-код в консоли браузера
2.Создать HTML-страницу со встроенным JavaScript
3.Подключить внешний файл script.js
4.Реализовать работу с переменными
5.Использовать условия и цикл
